PackWise AI — System Flowcharts

1. Authentication & Account Flow

%%{init: {"flowchart": {"curve": "linear", "nodeSpacing": 50, "rankSpacing": 60}} }%% flowchart TD A([Admin wants to create a new account]) --> B["Admin inputs Email, Name, Role, Password"] --> C["System hashes password & saves user to database"] --> D([PE / PM logs in]) --> E["PE/PM logs in with Email + Password"] E --> F{Are credentials valid?} F -- Yes --> G["Redirect to Dashboard"] F -- No --> H["Display error message"] H --> E G --> P{"Does user want to change password?"} P -- No --> L([Done – user is ready to use the system]) P -- Yes --> I["User goes to Settings → Change Password"] --> J["User inputs Current Password + New Password"] --> K["System updates password hash in database"] --> L E -. forgot password .-> M["User reports to Admin offline / via chat"] --> N["Admin searches user in User Management,\nclicks Reset Password"] --> O["Admin creates new Password & gives it to user"] --> E

2. Packing Engineer Flow (AI & Calculation)

%%{init: {"flowchart": {"curve": "linear", "nodeSpacing": 50, "rankSpacing": 60}} }%% flowchart TD A([PE starts a new plan]) --> B["PE creates New Packaging Plan & uploads prototype photo"] --> C["PE fills metadata checkboxes\n(Long Hair, Detachable Accessories, etc)"] --> D["YOLO model detects 4 joint targets:\nNeck, Waist, Wrist, Ankle"] --> E["System generates bounding boxes"] --> F["System calculates estimated cost\n(bounding boxes × strap price + additional costs)"] --> G["PE reviews bounding boxes & cost estimation on screen"] --> H{Is AI detection accurate?} H -- Yes --> I["PE clicks Submit"] H -- No --> J["PE manually adjusts:\nadd / delete / reposition boxes"] J --> G I --> K["System saves plan, status = Pending Approval"] --> L([Plan enters PM queue])

3. Product Manager Approval Flow

%%{init: {"flowchart": {"curve": "linear", "nodeSpacing": 50, "rankSpacing": 60}} }%% flowchart TD A([PM opens approval menu]) --> B["PM opens Pending Approvals menu"] --> C["PM selects the document submitted by PE"] --> D["PM reviews AI detection, metadata, & cost breakdown"] --> E{Is the plan approved?} E -- Approve --> F["PM clicks Approve"] --> G["Status = Finalized / Approved, data is locked"] --> H["Ready for report generation & production"] --> I([Done]) E -- Reject --> J["PM clicks Reject & inputs revision notes"] --> K["Status = Rejected / Need Revision"] --> L["PE revises plan & resubmits (back to Flow 2)"] -. resubmitted plan .-> D
